CMglGraphicManager::Init()

void Init( HWND hWnd, int nDispX, int nDispY, BOOL bFullscreen, D3DFORMAT formatTexture=D3DFMT_A8R8G8B8,
		DWORD dwD3dDeviceMode=D3D_DEVICE_MODE_DEFAULT );


概要

DirectX（のDirectGraphics）とこのクラスの初期化処理を行います。


引数

hWnd　- 親ウインドウのハンドル
nDispX　- 画面の幅
nDispY　- 画面の高さ
bFullscreen　- フルスクリーンにするか否か
formatTexture　- テクスチャのフォーマット（省略時 D3DFMT_A8R8G8B8）
dwD3dDeviceMode　- Direct3Dのデバイスモード（省略時 D3D_DEVICE_MODE_DEFAULT）
