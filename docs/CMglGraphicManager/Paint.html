CMglGraphicManager::Paint()

void Paint( RECT* rect, D3DCOLOR color );
void Paint( int left, int top, int right, int bottom, D3DCOLOR color );


概要

指定された矩形領域を塗り潰します。


引数

rect　- 塗り潰す矩形領域
top　- 塗り潰し開始位置X座標
left　- 塗り潰し開始位置Y座標
right　- 塗り潰し終了位置X座標
bottom　- 塗り潰し終了位置Y座標
color　- 塗り潰しとして使用する色（確かアルファブレンドも可…？）
