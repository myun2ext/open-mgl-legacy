概要

テキスト描画を行います。


引数

szText　-　描画するテキスト文字列（改行コードもサポートされる）
x　- 絵画開始位置X座標。省略時0
y　- 絵画開始位置Y座標。省略時0
size　- 文字のサイズ。省略時12
color　- 文字色。省略時黒
hFont　- FONTのハンドル


復帰値

なし
