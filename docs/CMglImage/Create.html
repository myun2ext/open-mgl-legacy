CMglImage::Create()

void Create( int x=0, int y=0, BOOL bRenderTarget=TRUE );
void Create( LPCSTR szFileName, BOOL bRenderTarget=TRUE, D3DCOLOR colorKey=DEF_COLORKEY );


概要

画像イメージの作成（初期化）を行います。
szFileName 指定版の場合は、画像ファイルより作成を行います。


引数

x　- 作成される画像の幅。省略時、または0指定時は画面サイズと同じ
y　- 作成される画像の高さ。省略時、または0指定時は画面サイズと同じ
bRenderTarget　- レンダリングターゲットとして使用するか否かのフラグ
szFileName　- 読み込む画像ファイル
colorKey　- 透過色として使用する色（カラーキー）


備考

szFileName にて指定可能な画像フォーマットは以下の通りです。

BMP, DDS, JPEG, PNG

また、bRenderTarget にTRUEを設定した場合、その画像データはVRAM上に配置されます。
逆に FALSE を設定した場合はシステムメモリ上に配置されます。
