CMglImage クラス　- 画像データ管理クラス

画像データの管理を行います。
このクラスのインスタンス一つが一枚の画像データに相当します。


メソッド

初期化/開放系メソッド：
＞Create()　- 画像データの構築/読み込み
＞Release()　- 開放（省略可能）

描画系メソッド：
＞Draw()　- 描画
＞FastDraw()　- 高速描画

その他のメソッド：
＞void SetRender();　- レンダリング対象をこの画像データに設定
＞void Clear();　- 画面のクリア（白色塗り潰し）
＞void Clear( D3DCOLOR color );　- 画面のクリア（塗り潰しの色指定）
＞void Paint( RECT* rect, D3DCOLOR color );　- 矩形領域の塗り潰し

設定/取得系メソッド：
＞int GetBmpWidth();　- BMPの幅取得
＞int GetBmpHeight();　- BMPの高さ取得
